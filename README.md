# Knots Project

This project is an early-stage data engineering exploration focused on knot theory. It currently consists of a SQLite database storing a small set of knot invariants sourced from KnotInfo, with an emphasis on structuring mathematical data for future computation and analysis.

I began this project with the goal of working toward the Knot-Ribbon and Volume Conjectures, but realized I need a deeper theoretical foundation first. I plan to expand this into a more complete pipeline for computing and organizing knot invariants as my understanding of the math develops.
